# 🎮 VibeCoding 빠른 시작 가이드

## 🚀 Git 없이 바로 시작하기!

### Windows 사용자
1. `start-workshop.bat` 더블클릭
2. 브라우저에서 자동으로 페이지가 열립니다
3. AI 룰을 생성하고 Cursor에 복사하세요

### Mac 사용자
1. `start-workshop.sh` 더블클릭
2. 브라우저에서 자동으로 페이지가 열립니다
3. AI 룰을 생성하고 Cursor에 복사하세요

## 📁 포함된 내용
- ✅ 모든 게임 템플릿
- ✅ AI 룰 생성기
- ✅ 5가지 교육 워크플로우
- ✅ 워크샵 진행 가이드
- ✅ 예제 게임들
